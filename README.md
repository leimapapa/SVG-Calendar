# SVG-Calendar
A lightweight, super-simple SVG Calendar that populates a single element

Minified version is only 2.616 kb!



## Dependencies

* None babay!


## Usage
Use it by pointing it to your element:

```javascript
svgCal(document.querySelector('#myElement'));
```
